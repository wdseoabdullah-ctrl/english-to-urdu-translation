# English to Urdu Translator Web App

A simple, fast, and user-friendly web application that helps users quickly access English to Urdu translation tools.

## 🌐 Live Demo
[https://wdseoabdullah-ctrl.github.io/english-to-urdu-translator/](https://github.com/wdseoabdullah-ctrl/english-to-urdu-translation)

## 📌 Features
- Clean and responsive design
- Quick access to translation tool
- Lightweight and fast loading
- Works on all devices (mobile & desktop)
- Simple user interface

## 🚀 How It Works
This app provides users with two main options:
- **Translate Now:** Instantly opens the English to Urdu translation tool
- **Visit Website:** Redirects to the main website for more resources
